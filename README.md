# Poona Club

Responsive Next.js portal for member verification, admin member management, WhatsApp OTP workflows, and future club services.

## Scripts

- `npm run dev`
- `npm run lint`
- `npm run build`
- `node scripts/import-members.mjs`

## Stack

- Next.js 16 App Router
- Tailwind CSS 4
- Supabase-ready schema and helpers
- TechUpServices WhatsApp delivery wrapper

## Environment

Copy `.env.example` to `.env.local` and provide the required values before connecting live Supabase and TechUpServices.

## Deployment

Production is served at `https://www.pclprofile.com`.
